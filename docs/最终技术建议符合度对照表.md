## 三层推理架构

```
Layer 1: 规则引擎 (score_style)           — 确定性匹配, 永远运行, 保证下限
Layer 2: Neo4j 知识图谱 (graph_matcher)   — 关系推理, 发现隐含关联, 增强关系
Layer 3: LLM 投票/摘要 (evaluation-agent) — 语义判断, 自然语言解释, 提升上限
```

融合策略: `final_score = rule_score + min(graph_score, rule_score // 2)`

## 异常处理矩阵

| 故障点 | 影响 | 处理策略 |
|--------|------|---------|
| Neo4j 不可用 | 图谱评分归零 | 自动 fallback JSON |
| LLM 不可用 | 无 LLM 摘要/投票 | 规则模板降级 |
| LangGraph 未安装 | 编排引擎降级 | 手动编排 fallback |
| ADR 写入失败 | ADR 状态=失败 | 不阻塞推荐主链路 |
| 缓存不可用 | 每次实时计算 | 缓存模块静默降级 |
| refactoring-agent 不可达 | 无重构建议 | 不阻塞推荐主链路 |
