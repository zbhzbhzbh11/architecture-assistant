# 项目实施计划（按周）

## 第1周：需求与架构基线
- 完成需求规格说明书初稿。
- 明确微服务边界和 Agent 协作流程。
- 定义 API 契约与数据结构。

## 第2周：核心能力开发
- 打通需求提取、架构匹配、评估生成主链路。
- 完成知识库初始化（10+架构风格）。
- 增加规则引擎可解释性输出。

## 第3周：可视化与测试
- 完成前端对比矩阵与拓扑图渲染。
- 维护 20+ 场景测试集并执行回归。
- 输出阶段性测试报告。

## 第4周：优化与答辩
- 接入真实 LLM 服务并做 Prompt 调优。
- 增加决策溯源与缓存策略。
- 完成答辩 PPT、演示脚本与彩排。
